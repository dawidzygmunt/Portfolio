# Portfolio

My portfolio website showcasing projects and skills. Responsive design with animations and a contact form.

## Demo

http://dawidzygmuntdev.pl/

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## Running Locally

```bash
npm install
npm run dev
```

The site will be available at `http://localhost:3000`.
